# NHTSA Safety Recall 24V-456

## Seat Belt Pretensioner May Not Deploy

**Affected Vehicles:** 2021-2022 Chevrolet Equinox
**Date Issued:** 2024-04-10
**Estimated Affected:** 180,000 vehicles

## Defect Description
Front seat belt pretensioners may not activate in a frontal collision due to a software calibration error in the restraints control module.

## Remedy
Dealers will update the restraints control module software. No hardware replacement needed. Repair time: 0.3 hours.

## Interim Driver Guidance
Seat belts still function as standard 3-point belts. Only the pretensioner (which pulls the belt tight in a crash) is affected.

## Driver Communication
> "Your vehicle is affected by a safety recall — 24V-456. Front seat belt pretensioners may not activate in a frontal collision due to a software calibration error in the restraints control module. The repair is free at any authorized dealer. Would you like me to help schedule an appointment?"
