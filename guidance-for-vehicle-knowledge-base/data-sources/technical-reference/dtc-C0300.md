# C0300 — Rear Steering Position Sensor

## Summary
Diagnostic Trouble Code C0300 indicates: rear steering position sensor.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **Position sensor failure** (45%))
2. **Wiring damage** (25%))
3. **Steering actuator** (20%))
4. **Module fault** (10%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Your rear steering sensor has failed. If your vehicle has four-wheel steering, it may be disabled. Drive carefully at low speeds and service promptly."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
