# P0010 — Intake Camshaft Position Actuator Circuit (Bank 1)

## Summary
Diagnostic Trouble Code P0010 indicates: intake camshaft position actuator circuit (bank 1).

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Faulty VVT solenoid** (45%))
2. **Wiring harness damage** (25%))
3. **Low oil pressure** (20%))
4. **PCM fault** (10%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "Your engine's variable valve timing system has a fault. You may notice slightly rough idle or reduced fuel economy. Safe to drive — schedule service within 1-2 weeks."

## Repair Estimate
$100-$1,500 (routine repair)
