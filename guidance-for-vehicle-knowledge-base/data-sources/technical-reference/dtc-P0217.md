# P0217 — Engine Coolant Over Temperature

## Summary
Diagnostic Trouble Code P0217 indicates: engine coolant over temperature.

## Severity: P0
**CRITICAL SAFETY ISSUE** — Do not drive. Arrange immediate service or tow.

## Common Causes (by likelihood)
1. **Coolant leak** (30%))
2. **Failed water pump** (25%))
3. **Stuck thermostat** (20%))
4. **Radiator blockage** (15%))
5. **Fan failure** (10%))

## Diagnostic Steps
1. Verify code with freeze frame data
2. Check related sensor wiring and connectors
3. Test sensor output with scan tool live data
4. Check for TSBs related to this code and vehicle
5. Perform component test per service manual

## Driver Communication
> "Your engine is overheating. Pull over safely and turn off the engine immediately. Do not open the radiator cap. Call for roadside assistance — driving further risks severe engine damage."

## Repair Estimate
$500-$3,000+ (safety-critical repairs)
