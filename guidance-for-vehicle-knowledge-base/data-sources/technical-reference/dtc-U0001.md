# U0001 — High Speed CAN Communication Bus

## Summary
Diagnostic Trouble Code U0001 indicates: high speed can communication bus.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **CAN bus wiring fault** (35%))
2. **Failed module pulling bus down** (25%))
3. **Connector corrosion** (20%))
4. **Termination resistor** (10%))
5. **Aftermarket device** (10%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Your vehicle's main communication network has a fault. Multiple systems may be affected. Remove any OBD-II plug-in devices and restart. If symptoms persist, service immediately."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
