# P2096 — Post Catalyst Fuel Trim System Too Lean (Bank 1)

## Summary
Diagnostic Trouble Code P2096 indicates: post catalyst fuel trim system too lean (bank 1).

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Exhaust leak before rear O2** (35%))
2. **Failing catalytic converter** (30%))
3. **Rear O2 sensor fault** (20%))
4. **Fuel system lean** (15%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "The system downstream of your catalytic converter is reading lean. This often indicates the catalyst is starting to fail or there's an exhaust leak. Schedule diagnosis within 1-2 weeks."

## Repair Estimate
$100-$1,500 (routine repair)
