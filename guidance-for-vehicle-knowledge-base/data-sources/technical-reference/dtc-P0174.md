# P0174 — System Too Lean (Bank 2)

## Summary
Diagnostic Trouble Code P0174 indicates: system too lean (bank 2).

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Vacuum leak** (40%))
2. **Dirty MAF sensor** (25%))
3. **Weak fuel pump** (15%))
4. **Clogged injector** (10%))
5. **Exhaust leak** (10%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "Your engine is running lean on Bank 2. Similar to P0171 but on the opposite cylinder bank. If both banks are lean, suspect a common cause like the MAF sensor or fuel pump."

## Repair Estimate
$100-$1,500 (routine repair)
