# P0442 — EVAP System Small Leak Detected

## Summary
Diagnostic Trouble Code P0442 indicates: evap system small leak detected.

## Severity: P3
**Monitor** — Address at next scheduled service.

## Common Causes (by likelihood)
1. **Loose gas cap** (50%))
2. **Cracked EVAP hose** (25%))
3. **Failed purge valve** (15%))
4. **Charcoal canister crack** (10%))

## Diagnostic Steps
1. Verify code is current (not historical)
2. Check for related codes
3. Inspect obvious items (fluid levels, filters, caps)
4. Clear code and monitor for return
5. Address at next scheduled service if code returns

## Driver Communication
> "A small leak was detected in your fuel vapor system. First, check that your gas cap is tight and clicks when closed. If the light returns after a few drive cycles, schedule service at your convenience."

## Repair Estimate
$50-$500 (minor or deferred)
