# B1517 — Seat Memory Module Fault

## Summary
Diagnostic Trouble Code B1517 indicates: seat memory module fault.

## Severity: P3
**Monitor** — Address at next scheduled service.

## Common Causes (by likelihood)
1. **Memory module failure** (40%))
2. **Seat position sensor** (30%))
3. **Wiring** (20%))
4. **Switch fault** (10%))

## Diagnostic Steps
1. Verify code is current (not historical)
2. Check for related codes
3. Inspect obvious items (fluid levels, filters, caps)
4. Clear code and monitor for return
5. Address at next scheduled service if code returns

## Driver Communication
> "Your seat memory system has a fault. The seat still adjusts manually but won't save/recall positions. Purely a convenience issue — fix whenever convenient."

## Repair Estimate
$50-$500 (minor or deferred)
