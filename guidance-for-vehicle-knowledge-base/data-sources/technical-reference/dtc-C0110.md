# C0110 — Pump Motor Circuit Malfunction

## Summary
Diagnostic Trouble Code C0110 indicates: pump motor circuit malfunction.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **ABS pump motor failure** (40%))
2. **Relay fault** (25%))
3. **Wiring** (20%))
4. **ABS module** (15%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Your ABS hydraulic pump has a fault. ABS will not function in an emergency stop. Normal braking is unaffected. Service within 48 hours — avoid aggressive driving."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
