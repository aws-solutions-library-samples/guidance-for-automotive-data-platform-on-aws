# P0500 — Vehicle Speed Sensor Malfunction

## Summary
Diagnostic Trouble Code P0500 indicates: vehicle speed sensor malfunction.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **Failed VSS** (45%))
2. **Wiring damage** (25%))
3. **Connector corrosion** (15%))
4. **Transmission internal fault** (15%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Your vehicle speed sensor has failed. The speedometer may not work, and the transmission may not shift correctly. Drive carefully to service — the transmission may stay in one gear."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
