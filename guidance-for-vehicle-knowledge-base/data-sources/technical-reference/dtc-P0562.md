# P0562 — System Voltage Low

## Summary
Diagnostic Trouble Code P0562 indicates: system voltage low.

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Failing alternator** (40%))
2. **Corroded battery terminals** (25%))
3. **Worn drive belt** (20%))
4. **Parasitic drain** (15%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "Your charging system voltage is low. The battery may not be charging properly. If you notice dim lights or slow cranking, get this checked soon — you could end up stranded if the battery dies."

## Repair Estimate
$100-$1,500 (routine repair)
