# C0161 — Brake System Pressure Circuit

## Summary
Diagnostic Trouble Code C0161 indicates: brake system pressure circuit.

## Severity: P0
**CRITICAL SAFETY ISSUE** — Do not drive. Arrange immediate service or tow.

## Common Causes (by likelihood)
1. **Brake fluid leak** (35%))
2. **Master cylinder failure** (30%))
3. **Pressure sensor fault** (20%))
4. **ABS modulator leak** (15%))

## Diagnostic Steps
1. Verify code with freeze frame data
2. Check related sensor wiring and connectors
3. Test sensor output with scan tool live data
4. Check for TSBs related to this code and vehicle
5. Perform component test per service manual

## Driver Communication
> "Critical brake pressure loss detected. Pull over immediately. Check brake fluid level. If pedal feels soft or goes to the floor, do NOT drive — call for a tow immediately."

## Repair Estimate
$500-$3,000+ (safety-critical repairs)
