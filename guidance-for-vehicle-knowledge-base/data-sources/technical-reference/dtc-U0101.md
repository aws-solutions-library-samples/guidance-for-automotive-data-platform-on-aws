# U0101 — Lost Communication with TCM

## Summary
Diagnostic Trouble Code U0101 indicates: lost communication with tcm.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **TCM failure** (30%))
2. **CAN wiring to TCM** (25%))
3. **TCM power/ground** (20%))
4. **Connector** (15%))
5. **Software fault** (10%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Communication with your transmission control module is lost. The transmission may default to a single gear. Drive gently and directly to service."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
