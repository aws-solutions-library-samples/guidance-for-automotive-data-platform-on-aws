# P0299 — Turbocharger/Supercharger Underboost

## Summary
Diagnostic Trouble Code P0299 indicates: turbocharger/supercharger underboost.

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Boost leak in intercooler piping** (35%))
2. **Wastegate stuck open** (25%))
3. **Failed boost pressure sensor** (20%))
4. **Turbo bearing wear** (15%))
5. **Clogged air filter** (5%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "Your turbocharger isn't producing full boost pressure. You'll notice reduced power, especially at higher speeds or under load. Safe to drive gently — avoid heavy acceleration until serviced."

## Repair Estimate
$100-$1,500 (routine repair)
