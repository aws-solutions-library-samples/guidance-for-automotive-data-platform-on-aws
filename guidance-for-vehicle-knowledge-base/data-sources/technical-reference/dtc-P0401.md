# P0401 — EGR Flow Insufficient

## Summary
Diagnostic Trouble Code P0401 indicates: egr flow insufficient.

## Severity: P3
**Monitor** — Address at next scheduled service.

## Common Causes (by likelihood)
1. **Carbon buildup in EGR valve** (50%))
2. **Failed EGR valve** (25%))
3. **Clogged EGR passages** (15%))
4. **EGR position sensor fault** (10%))

## Diagnostic Steps
1. Verify code is current (not historical)
2. Check for related codes
3. Inspect obvious items (fluid levels, filters, caps)
4. Clear code and monitor for return
5. Address at next scheduled service if code returns

## Driver Communication
> "Your EGR (exhaust gas recirculation) system isn't flowing enough. No driveability impact in most cases. May cause a slight knock under load. Address at next scheduled service."

## Repair Estimate
$50-$500 (minor or deferred)
