# U0401 — Invalid Data Received from ECM/PCM

## Summary
Diagnostic Trouble Code U0401 indicates: invalid data received from ecm/pcm.

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **ECM software fault** (30%))
2. **Intermittent CAN fault** (25%))
3. **ECM hardware degradation** (20%))
4. **Voltage spike damage** (15%))
5. **Connector** (10%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "Another module is receiving garbled data from your engine computer. This may cause intermittent issues. Often resolved with an ECM software update. Schedule service within a week."

## Repair Estimate
$100-$1,500 (routine repair)
