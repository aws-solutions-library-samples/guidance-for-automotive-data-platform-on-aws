# U0121 — Lost Communication with ABS Module

## Summary
Diagnostic Trouble Code U0121 indicates: lost communication with abs module.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **ABS module failure** (30%))
2. **CAN wiring fault** (25%))
3. **Power supply to ABS** (20%))
4. **Connector corrosion** (15%))
5. **Ground fault** (10%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Communication with your ABS module is lost. ABS, traction control, and stability control are all disabled. Normal brakes work. Service within 48 hours."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
