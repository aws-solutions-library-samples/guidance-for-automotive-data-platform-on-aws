# P0016 — Crankshaft/Camshaft Position Correlation (Bank 1 Sensor A)

## Summary
Diagnostic Trouble Code P0016 indicates: crankshaft/camshaft position correlation (bank 1 sensor a).

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **Stretched timing chain** (40%))
2. **Failed VVT actuator** (25%))
3. **Cam/crank sensor fault** (20%))
4. **Oil flow restriction** (15%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Your engine's timing is out of sync. You may hear a rattle on cold start or notice reduced power. Service within 48 hours — continued driving risks engine damage."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
