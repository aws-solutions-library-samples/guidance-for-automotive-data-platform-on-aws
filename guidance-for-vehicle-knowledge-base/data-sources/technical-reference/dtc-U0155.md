# U0155 — Lost Communication with Instrument Cluster

## Summary
Diagnostic Trouble Code U0155 indicates: lost communication with instrument cluster.

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Cluster failure** (35%))
2. **CAN wiring** (25%))
3. **Connector behind cluster** (20%))
4. **Power supply** (10%))
5. **Software** (10%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "Your instrument cluster has lost communication. Gauges may not display correctly. You won't have speed/fuel/temp readings. Drive carefully using GPS for speed and service soon."

## Repair Estimate
$100-$1,500 (routine repair)
