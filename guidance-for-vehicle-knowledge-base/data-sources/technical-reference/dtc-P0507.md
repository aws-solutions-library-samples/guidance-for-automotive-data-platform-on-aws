# P0507 — Idle Control System RPM Higher Than Expected

## Summary
Diagnostic Trouble Code P0507 indicates: idle control system rpm higher than expected.

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Vacuum leak** (35%))
2. **Dirty throttle body** (30%))
3. **Failed IAC valve** (20%))
4. **Intake gasket leak** (15%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "Your engine is idling too fast. This is usually a vacuum leak or dirty throttle body. Not dangerous, but the high idle wastes fuel. Schedule a cleaning/inspection within 1-2 weeks."

## Repair Estimate
$100-$1,500 (routine repair)
