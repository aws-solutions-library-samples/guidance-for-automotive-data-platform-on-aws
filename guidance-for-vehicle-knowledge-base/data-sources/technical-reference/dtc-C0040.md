# C0040 — Right Front Wheel Speed Sensor Circuit

## Summary
Diagnostic Trouble Code C0040 indicates: right front wheel speed sensor circuit.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **Damaged sensor wiring** (40%))
2. **Failed sensor** (30%))
3. **Corroded connector** (15%))
4. **Tone ring damage** (10%))
5. **Wheel bearing wear** (5%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Your right front ABS sensor has failed. ABS and stability control are disabled. Normal brakes still work. Avoid wet/icy conditions and service within 48 hours."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
