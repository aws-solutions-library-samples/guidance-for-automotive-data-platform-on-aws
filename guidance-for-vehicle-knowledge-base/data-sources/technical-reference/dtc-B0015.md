# B0015 — Passenger Frontal Airbag Deployment Control

## Summary
Diagnostic Trouble Code B0015 indicates: passenger frontal airbag deployment control.

## Severity: P0
**CRITICAL SAFETY ISSUE** — Do not drive. Arrange immediate service or tow.

## Common Causes (by likelihood)
1. **Airbag module connector fault** (35%))
2. **Passenger seat sensor** (25%))
3. **Airbag control module** (20%))
4. **Wiring harness** (20%))

## Diagnostic Steps
1. Verify code with freeze frame data
2. Check related sensor wiring and connectors
3. Test sensor output with scan tool live data
4. Check for TSBs related to this code and vehicle
5. Perform component test per service manual

## Driver Communication
> "Your passenger airbag system has a fault — it may not deploy in a crash. Do not allow passengers in the front seat until repaired. Schedule immediate service or arrange a tow."

## Repair Estimate
$500-$3,000+ (safety-critical repairs)
