# P0030 — HO2S Heater Control Circuit (Bank 1 Sensor 1)

## Summary
Diagnostic Trouble Code P0030 indicates: ho2s heater control circuit (bank 1 sensor 1).

## Severity: P3
**Monitor** — Address at next scheduled service.

## Common Causes (by likelihood)
1. **Failed O2 sensor heater** (60%))
2. **Blown fuse** (20%))
3. **Wiring open/short** (15%))
4. **PCM driver fault** (5%))

## Diagnostic Steps
1. Verify code is current (not historical)
2. Check for related codes
3. Inspect obvious items (fluid levels, filters, caps)
4. Clear code and monitor for return
5. Address at next scheduled service if code returns

## Driver Communication
> "An oxygen sensor heater has failed. No driveability impact — you may see slightly higher fuel consumption. Address at next scheduled service."

## Repair Estimate
$50-$500 (minor or deferred)
