# B1325 — Battery Voltage Out of Range

## Summary
Diagnostic Trouble Code B1325 indicates: battery voltage out of range.

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Weak battery** (40%))
2. **Alternator undercharging** (30%))
3. **Parasitic drain** (20%))
4. **BCM fault** (10%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "Your body control module is seeing abnormal battery voltage. This can cause various electrical glitches. Have the battery and charging system tested within a week."

## Repair Estimate
$100-$1,500 (routine repair)
