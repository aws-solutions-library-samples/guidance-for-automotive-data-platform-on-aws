# P0131 — O2 Sensor Circuit Low Voltage (Bank 1 Sensor 1)

## Summary
Diagnostic Trouble Code P0131 indicates: o2 sensor circuit low voltage (bank 1 sensor 1).

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Failed O2 sensor** (45%))
2. **Vacuum leak** (25%))
3. **Exhaust leak before sensor** (20%))
4. **Wiring fault** (10%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "An oxygen sensor is reading low, which may cause your engine to run rich. You might notice higher fuel consumption or a slight sulfur smell. Schedule service within 1-2 weeks."

## Repair Estimate
$100-$1,500 (routine repair)
