# P0455 — EVAP System Large Leak Detected

## Summary
Diagnostic Trouble Code P0455 indicates: evap system large leak detected.

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Missing/damaged gas cap** (40%))
2. **Disconnected EVAP hose** (30%))
3. **Failed vent valve** (15%))
4. **Cracked charcoal canister** (15%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "A significant leak in your fuel vapor system was detected. Check your gas cap first — if it's missing or damaged, replace it. If the cap is fine, schedule service within a week. You may smell fuel vapor."

## Repair Estimate
$100-$1,500 (routine repair)
