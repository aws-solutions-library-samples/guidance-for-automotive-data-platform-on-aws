# U1000 — Manufacturer Specific CAN Fault

## Summary
Diagnostic Trouble Code U1000 indicates: manufacturer specific can fault.

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Module-specific fault** (35%))
2. **CAN bus noise** (25%))
3. **Aftermarket device interference** (20%))
4. **Wiring** (10%))
5. **Software update needed** (10%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "A manufacturer-specific network fault has been detected. This requires dealer-level diagnostics with the OEM scan tool. Schedule service within 1-2 weeks."

## Repair Estimate
$100-$1,500 (routine repair)
