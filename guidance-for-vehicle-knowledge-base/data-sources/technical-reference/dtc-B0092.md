# B0092 — Left Headlamp Leveling Actuator Circuit

## Summary
Diagnostic Trouble Code B0092 indicates: left headlamp leveling actuator circuit.

## Severity: P3
**Monitor** — Address at next scheduled service.

## Common Causes (by likelihood)
1. **Failed leveling motor** (50%))
2. **Wiring fault** (25%))
3. **Control module** (15%))
4. **Sensor fault** (10%))

## Diagnostic Steps
1. Verify code is current (not historical)
2. Check for related codes
3. Inspect obvious items (fluid levels, filters, caps)
4. Clear code and monitor for return
5. Address at next scheduled service if code returns

## Driver Communication
> "Your left headlamp auto-leveling isn't working. The headlight will stay at its current position. Not a safety emergency but may affect visibility on hills. Fix at next service."

## Repair Estimate
$50-$500 (minor or deferred)
