# P0101 — Mass Air Flow Sensor Range/Performance

## Summary
Diagnostic Trouble Code P0101 indicates: mass air flow sensor range/performance.

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Dirty MAF sensor** (50%))
2. **Air filter restriction** (20%))
3. **Intake leak after MAF** (20%))
4. **Failed MAF** (10%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "Your mass airflow sensor isn't reading correctly. You might notice hesitation or poor fuel economy. Try cleaning the sensor first — if that doesn't help, schedule service within a week."

## Repair Estimate
$100-$1,500 (routine repair)
