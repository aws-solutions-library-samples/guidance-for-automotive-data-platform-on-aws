# P0128 — Coolant Thermostat Below Regulating Temperature

## Summary
Diagnostic Trouble Code P0128 indicates: coolant thermostat below regulating temperature.

## Severity: P3
**Monitor** — Address at next scheduled service.

## Common Causes (by likelihood)
1. **Stuck-open thermostat** (70%))
2. **Coolant temp sensor fault** (20%))
3. **Low coolant level** (10%))

## Diagnostic Steps
1. Verify code is current (not historical)
2. Check for related codes
3. Inspect obvious items (fluid levels, filters, caps)
4. Clear code and monitor for return
5. Address at next scheduled service if code returns

## Driver Communication
> "Your engine isn't reaching normal operating temperature. This usually means the thermostat is stuck open. Not urgent — your heater may blow lukewarm air. Fix at next service."

## Repair Estimate
$50-$500 (minor or deferred)
