# P0301 — Cylinder 1 Misfire Detected

## Summary
Diagnostic Trouble Code P0301 indicates: cylinder 1 misfire detected.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **Faulty spark plug** (35%))
2. **Failed ignition coil** (30%))
3. **Clogged fuel injector** (20%))
4. **Low compression** (10%))
5. **Vacuum leak at intake runner** (5%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Cylinder 1 is misfiring. If your check engine light is flashing, reduce speed and avoid hard acceleration. A steady light means you can drive to service, but schedule it within 48 hours."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
