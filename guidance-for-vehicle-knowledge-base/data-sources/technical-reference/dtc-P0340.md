# P0340 — Camshaft Position Sensor Circuit

## Summary
Diagnostic Trouble Code P0340 indicates: camshaft position sensor circuit.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **Failed cam sensor** (50%))
2. **Wiring damage** (25%))
3. **Connector corrosion** (15%))
4. **PCM fault** (10%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Your camshaft position sensor has failed. The engine may run rough, stall, or not start. If it's running, drive directly to service. If it won't start, you'll need a tow."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
