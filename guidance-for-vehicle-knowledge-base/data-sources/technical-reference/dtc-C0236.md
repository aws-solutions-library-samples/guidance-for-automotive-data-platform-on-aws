# C0236 — Rear Axle Differential Fluid Pressure Sensor

## Summary
Diagnostic Trouble Code C0236 indicates: rear axle differential fluid pressure sensor.

## Severity: P2
**Service within 1-2 weeks** — Vehicle is drivable but issue should be addressed.

## Common Causes (by likelihood)
1. **Sensor failure** (45%))
2. **Low differential fluid** (30%))
3. **Wiring** (15%))
4. **Differential seal leak** (10%))

## Diagnostic Steps
1. Read freeze frame data for conditions during fault
2. Check for related codes that may indicate root cause
3. Inspect wiring and connectors in affected circuit
4. Test component operation with scan tool
5. Clear code and road test to verify repair

## Driver Communication
> "Your rear differential pressure sensor has flagged an issue. Check differential fluid level. If fluid is low, there may be a leak. Schedule service within 1-2 weeks."

## Repair Estimate
$100-$1,500 (routine repair)
