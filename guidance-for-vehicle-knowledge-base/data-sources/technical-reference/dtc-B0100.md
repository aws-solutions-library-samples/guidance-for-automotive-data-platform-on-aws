# B0100 — HVAC Blower Motor Control Circuit

## Summary
Diagnostic Trouble Code B0100 indicates: hvac blower motor control circuit.

## Severity: P3
**Monitor** — Address at next scheduled service.

## Common Causes (by likelihood)
1. **Failed blower motor resistor** (45%))
2. **Blower motor failure** (30%))
3. **Wiring fault** (15%))
4. **Control module** (10%))

## Diagnostic Steps
1. Verify code is current (not historical)
2. Check for related codes
3. Inspect obvious items (fluid levels, filters, caps)
4. Clear code and monitor for return
5. Address at next scheduled service if code returns

## Driver Communication
> "Your cabin blower motor has a fault. You may lose some or all fan speeds for heating/cooling. Not a safety issue but affects comfort. Schedule at your convenience."

## Repair Estimate
$50-$500 (minor or deferred)
