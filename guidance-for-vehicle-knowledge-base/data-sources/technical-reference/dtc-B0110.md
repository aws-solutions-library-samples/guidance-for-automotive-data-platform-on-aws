# B0110 — Driver Door Lock Actuator Circuit

## Summary
Diagnostic Trouble Code B0110 indicates: driver door lock actuator circuit.

## Severity: P3
**Monitor** — Address at next scheduled service.

## Common Causes (by likelihood)
1. **Failed lock actuator** (50%))
2. **Wiring break in door harness** (25%))
3. **BCM fault** (15%))
4. **Connector corrosion** (10%))

## Diagnostic Steps
1. Verify code is current (not historical)
2. Check for related codes
3. Inspect obvious items (fluid levels, filters, caps)
4. Clear code and monitor for return
5. Address at next scheduled service if code returns

## Driver Communication
> "Your driver's door power lock isn't working electrically. You can still lock/unlock manually with the key. Fix at your convenience — not a safety concern."

## Repair Estimate
$50-$500 (minor or deferred)
