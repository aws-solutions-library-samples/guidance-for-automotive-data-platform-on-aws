# U0073 — Control Module Communication Bus Off

## Summary
Diagnostic Trouble Code U0073 indicates: control module communication bus off.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **Bus short to ground** (30%))
2. **Failed module** (25%))
3. **Water intrusion** (20%))
4. **Wiring damage** (15%))
5. **Connector fault** (10%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "A communication bus has gone offline. Multiple warning lights may be on. The vehicle may enter limp mode. Drive directly to service if drivable, otherwise arrange a tow."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
