# P0715 — Input/Turbine Speed Sensor Circuit

## Summary
Diagnostic Trouble Code P0715 indicates: input/turbine speed sensor circuit.

## Severity: P1
**Service within 48 hours** — Safety system may be degraded.

## Common Causes (by likelihood)
1. **Failed speed sensor** (45%))
2. **Wiring damage** (25%))
3. **Connector issue** (15%))
4. **Internal transmission damage** (15%))

## Diagnostic Steps
1. **Verify safety** — is the vehicle safe to drive?
2. Read freeze frame and all stored codes
3. Check for obvious damage (wiring, leaks, physical)
4. Test affected system with scan tool bidirectional controls
5. Follow manufacturer diagnostic tree

## Driver Communication
> "Your transmission's input speed sensor has failed. Shifts may be harsh, delayed, or the transmission may not shift at all. Drive gently and directly to service."

## Repair Estimate
$200-$2,000 (urgent but not emergency)
